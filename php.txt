					   - Nếu có kế thừa: - Class parent: $this là đối tượng đang gọi nó, static là đối tượng gọi nó, self là chính nó 
					   					 - Class children: $this là chính nó, static là chính nó, self là chính nó, parent là Class parent 
					   - Trả về object this không cần dùng new, nhưng self và static phải cần dùng new 

	+ Overwrite: Nếu muốn overwrite một function thì phải cùng phương thức và tham số 

	+ Overloading: Overwrite lên các function tương tác mặc định với đối tượng của php __set(), __get(), __isset(), __unset(), __call(), __callStatic() 

	+ Early binding vs Late binding: 
									Nếu B kế thừa A: 
									- A dùng trỏ $this hoặc static thì khi B gọi A, đối tượng thực hiện lệnh gọi là B (đối tượng hiện tại) Early binding 
									- A dùng trỏ self thì khi B gọi A, thì đối tượng thực hiện lệnh gọi là A (đối tượng khởi tạo) Late binding 
									class A {
									    
									    public function X(){
									        print_r('A');    
									    }
									    
									    public function Y(){ 
									        $this->X(); 	// B 
									        static::X(); 	// B
									        self::X(); 		// A 
									    } 
									    
									} 
									class B extends A {
										
										public function X(){
										    print_r('B'); 
										}  
										
										public function Y(){
											
											$this->X(); 
											parent::Y(); 
									    } 
									    
									}
